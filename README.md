# Menstrual Health Awareness Game

This is a modern, interactive educational web application based on the classic Snakes & Ladders board game.

## What's Included

- The Menstrual Health Awareness game board with player avatars, dice, and audio assets
- The original game mechanics have been enhanced into a modern web application:
  - `index.html`
  - `css/styles.css`
  - `js/game.js`
  - `assets/images/*`
  - `assets/audio/*`
- The game behavior teaches menstrual health literacy through interactive questions: players must roll 6 to start, dice rolls animate, snakes and ladders trigger true/false questions, names can be entered before play, and the winner gets a modal.
- Enhanced feature: before a player enters the board, they answer a menstrual health question. A correct answer gives that entry roll a 65% chance of becoming a 6.

## Key Features

- Dynamic question loading from `questions.md`
- True/False menstrual health questions
- Difficulty-based question categorization (easy, medium, hard)
- Light/Dark theme toggle
- Sound effects and background music controls
- Interactive rules carousel
- Responsive design for all devices
- Accessibility features including read-aloud support
