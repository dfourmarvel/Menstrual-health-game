## 💡 Customization Examples

### Add a Question
Edit `js/game.js`, find questions array, add:
```javascript
{
  question: "Your question here?",
  answer: true  // or false
}
```

### Change a Color
Edit `css/styles.css`, find :root section, change:
```css
--color-primary: #new-color;
```

### Change Game Rules
Edit `js/game.js`, find gameRules array, modify text

### Adjust Speech Speed
Edit `js/game.js`, find speechManager, change:
```javascript
this.utterance.rate = 0.9;  // Change 0.9 to desired speed
```
