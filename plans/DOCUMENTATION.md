# Consolidated Project Documentation

## 📖 Overview
This repository contains the **Menstrual Health Awareness Game**, a modern, accessible, web‑based Snakes & Ladders style educational game. The current version (2.0) includes a full redesign, light/dark theme, audio controls, and a WCAG‑AA compliant **Read‑Aloud (TTS)** feature.

## 🚀 Quick Start
1. Open `index.html` in a browser to play the game.
2. Review the concise player guide in **README.md** for rules.
3. For a high‑level project snapshot, read **PROJECT_STATUS.md** (≈9 k words).
4. For a short summary of the Read‑Aloud feature, see **READ_ALOUD_SUMMARY.md**.
