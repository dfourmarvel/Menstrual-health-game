## 🚀 How to Use

### Playing the Game
1. Open `index.html` in web browser
2. Read the rules (carousel format)
3. Enter player names
4. Roll the dice and move tokens
5. Answer health questions
6. First to 100 wins!

### Customizing Your Experience
- **Change Theme:** Click moon/sun icon in navbar
- **Toggle Music:** Click music note icon
- **Toggle Sound:** Click speaker icon
- **Toggle Read Aloud:** Click speaker icon (top right)
- **Settings:** All preferences automatically saved

### Mobile Play
- Game fully responsive on phones and tablets
- Touch-friendly buttons and controls
- Optimized layout for small screens
- Portrait and landscape supported
